Volcanic Eruptions Patterns in the Americas

This project analyzes historical volcanic activity across North and South America using real eruption and volcano data from the Smithsonian Global Volcanism Program. Using Python, Pandas, Seaborn, Matplotlib, and Basemap, I performed geospatial and statistical analysis to explore eruption frequency, volcano types, population exposure, and elevation trends. The results are visualized through maps, pie charts, scatter plots, and regression analysis in a Jupyter Notebook.

Stock Value Analysis: BA and JPM

This project analyzes the historical stock performance of Boeing (BA) and JPMorgan Chase (JPM) using a combination of SQL, Python, and Tableau. SQL was used to calculate daily price differences and return percentages across a five-year dataset. In Python, I applied the ARIMA time series forecasting model using Pandas, Statsmodels, and Matplotlib to predict the closing price 30 days into the future. Finally, the results were visualized in Tableau through a dynamic dashboard highlighting key financial metrics, sector comparisons, and forecasted trends.
